# HorseRacing

## 주제: 경주마 레이싱 프로그램

## 요구사항
1. 경주마의 정보는 이름, 등급, 체력, 스피드가 있다.
2. 이용할 경주마들을 선택할 수 있다.
3. 경주를 통해 상금을 획득한다.
4. 상금을 통해 경주마를 뽑는다.
5. 경주마의 등급과 스피드는 랜덤으로 정해진다.
6. 경주가 끝나고 체력이 일정 감소한다.
7. 체력이 60이상만 경주가 가능하다.
8. 휴식을 통해 체력을 회복한다.
